# SimSense: A Real-Time Depth Sensor Simulator
This is a GPU-accelerated depth sensor simulator for python, implemented with CUDA. The goal of this project is to provide a real-time, highly adjustable, and easy-to-use module that can be utilized to simulate different kinds of depth sensors in the real world. For more information, please refer to README in `main` branch.

## Usage In SAPIEN
An example usage of the module within SAPIEN can be found at [https://sapien.ucsd.edu/docs/latest/tutorial/rendering/raytracing_renderer.html#](https://sapien.ucsd.edu/docs/latest/tutorial/rendering/raytracing_renderer.html#).
